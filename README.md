<details>
<summary><b>🤖 Code Optimizer</b> — Multi-agent LangGraph system for Python code optimization</summary>
<br>

Analyzes Python code, generates optimized versions, validates improvements, and produces detailed optimization reports — built on a **multi-agent LangGraph** architecture.

**How it works:**
- Agent 1 analyzes the input code for bottlenecks
- Agent 2 generates optimized alternatives using Gemini with Ollama fallback
- Agent 3 validates correctness and benchmarks improvements
- FAISS-powered retrieval surfaces relevant optimization patterns

**Stack:** `LangGraph` `Gemini` `Ollama` `FAISS` `Docker` `Flask` `React`

</details>

<details>
<summary><b>🔍 GitHub Repo Assistant</b> — RAG-powered Q&A for any GitHub repository</summary>
<br>

Understands and answers questions about any GitHub repository using **semantic search, vector databases, and Gemini AI** — making it easy to explore unfamiliar codebases without reading file by file.

**Pipeline:**
```
Clone repo → Preprocess files → Language-aware chunking
→ HuggingFace Embeddings → ChromaDB storage
→ MMR retrieval + metadata filtering → Gemini answer generation
```

**What I learned:** RAG architecture, vector databases, embedding pipelines, semantic retrieval, and production-level AI workflow design.

**Stack:** `Streamlit` `LangChain` `ChromaDB` `HuggingFace Embeddings` `GitPython` `PyGithub` `Gemini`

</details>
